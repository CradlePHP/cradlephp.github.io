Sink - Overview

The kitchen sink briefly shows how Cradle can be used as full stack solution. It's purpose is to provide some context and examples on the usage of the micro framework. The sink has the following features and uses the following technologies.

Features

  • Full Stack MVC Architecture
  • Queue Centric Events
  • CSRF protection
  • Internationalization (i18n / l10n)
  • Customizable CLI commands

Technologies

  • Queues with RabbitMQ
  • Index Searching with ElasticSearch
  • Data Caching with Redis
  • Mailing with SwiftMailer
  • Templating with Handlebars
  • Captcha with reCaptcha
  • Server and Client Side CDN uploads with AWS S3
